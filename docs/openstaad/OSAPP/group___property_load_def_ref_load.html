const VARIANT FAR &  varLoadCaseTitle,
const long  nLoadType

Creates a new Reference Load Definition and set as active.

Parameters
[in]nLoadNo(Reference) load case reference ID to be assigned to new created reference load case.
[in]varLoadCaseTitleLoad case string title.
[in]nLoadTypeType of the load.
Value Load Type Value Load Type

1 Live 13 Imperfection

' Create Reference Load Definition, with Reference Load Definition(s) number ID of 1, Loading Type of Dead Load and title of "REF LOAD CASE 1".
