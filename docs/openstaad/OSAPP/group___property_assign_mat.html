Assign material to member.

Parameters
[in]strMaterialNameIdentification title of material.
[in]varMemberNoSingle or an array of intergers containing list of member nos.
Return values
0OK.
-3001Cannot find member varMemberNo.
-6023Material not found.
C++ Syntax
