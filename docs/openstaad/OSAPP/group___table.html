)
private

Add or create a report data table and returns the table no for a specified table.

Parameters
[in]ToReportNoReport Table No.
[in]szTableNameReport Table Name.
[in]RowNosNo of rows for specified table.
[in]ColNosNo of columns for specified table.
C++ Syntax

VARIANT TableNo = OSTableUI::AddTable(1, "Table1", 10, 5);
VBA Syntax
' Create a report "Table1" of #10 rows and #5 columns.
Dim TableNo As VARIANT = OSTableUI.AddTable(1, "Table1", 10, 5)

void OSTableUI::SetCellTextColor ( const VARIANT FAR &  ReportNo,

const VARIANT FAR &  lBlueValue 
)
private

Sets the color of the text to be displayed in a particular cell. By default, the color is always set to black.

Parameters
[in]ReportNoReport Table No.
[in]TableNoReport Table No.
[in]lRowNo of rows for specified table.
[in]lColNo of columns for specified table.
[in]lRedValueAn integer between 0 and 255 that represents the intensity of red in the color for the text.
[in]lGreenValueAn integer between 0 and 255 that represents the intensity of green in the color for the text.
[in]lBlueValueAn integer between 0 and 255 that represents the intensity of blue in the color for the text.
C++ Syntax
// An integer between 0 and 255 that represents the intensity of red, green or blue in the color for the text.
// A value of 0 for a particular color indicates that no shade of that color will be mixed into the final color.
// Sets the color (Green) of the text to be displayed in a particular cell.
OSTableUI::SetCellTextColor(1, TableNo, 10, 5, 0, 255, 0);
VBA Syntax
' An integer between 0 and 255 that represents the intensity of red, green or blue in the color for the text.
' A value of 0 for a particular color indicates that no shade of that color will be mixed into the final color.
' Sets the color (Green) of the text to be displayed in a particular cell.
OSTableUI.SetCellTextColor(1, TableNo, 10, 5, 0, 255, 0)
