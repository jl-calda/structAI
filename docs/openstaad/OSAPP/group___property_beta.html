VARIANT OSPropertyUI::AssignBetaAngle ( const VARIANT FAR &  varnBeamNo,
const VARIANT FAR &  varfBetaAngle 
)

Assign beta angle to beam(s).

Parameters
[in]varnBeamNoThe beam number ID(s) of VARIANT array.
[in]varfBetaAngleThe beta angle in degrees.
Return values
0OK.
-1061 dimensional array of long expected.
-6009Library Error: Unable to assign BETA angle.
Example
// Assign beta angle 90.0 to beam #1, 2, 3, 4.
VARIANT RetVal = OSPropertyUI::AssignBetaAngle(varnBeamNo, double varfBetaAngle = 90.0);
VBA Syntax
' Assign beta angle 90.0 to beam #1, 2, 3, 4.
Dim RetVal As VARIANT = OSPropertyUI.AssignBetaAngle(varnBeamNo, 90.0)
