VARIANT OSLoadUI::AddAutoLoadCombinations ( const VARIANT FAR &  varCode,
const VARIANT FAR &  varCategory,
const VARIANT FAR &  varLoadList,
VARIANT FAR &  varStartLoadCaseNo 
)

Automatically adds load combination.

Parameters
[in]varCodeLoad Combination Code string name (refer to "AutoLoadComb.ini").
[in]varCategoryLoad Combination Category string name (refer to corresponding rule ini file defined in "AutoLoadComb.ini").
[in]varLoadListLoad case reference ID(s) VARIANT array.
[out]varStartLoadCaseNo(Combination) load case reference ID with which automatically load combination generates starting.
Return values
0OK.
C++ Syntax

VARIANT RetVal = OSLoadUI::AddAutoLoadCombinations((LPCTSTR)"AISC", (LPCTSTR)"GENERAL", varLoadList, &varStartLoadCaseNo);
VBA Syntax
' Automatically add load combination using AISC code GENERAL category.
Dim RetVal As VARIANT = OSLoadUI.AddAutoLoadCombinations("AISC", (LPCTSTR)"GENERAL", varLoadList, &varStartLoadCaseNo)
