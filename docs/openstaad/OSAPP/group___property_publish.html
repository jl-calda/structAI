Get project published name by STAAD profile name.

Parameters
[in]strSTAADNameSTAAD profile name
[in]varnCountryThe country CODE.
Return values
<BSTR>Published profile name
NULLUnable to find any equivalent published name corresponding to STAAD profile name strSTAADName.
C++ Syntax

VBA Syntax
