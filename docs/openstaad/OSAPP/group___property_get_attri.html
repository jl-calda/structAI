long OSPropertyUI::GetAssignedAttributeCount ( long  nMembOrPlateNo)

Get the number of attribute with nMembOrPlateNo member or plate included.

Parameters
[in]nMembOrPlateNoThe number ID of member or plate.
Returns
The number of attribute associated with nMembOrPlateNo member (if member with this ID exists) or plate (if member not exist).
