VARIANT OSGeometryUI::GetParametricSurfaceCount ( )
private

Returns the number of surfaced defined with parametric model.

Returns
The number of meshed surface.
C++ Syntax
// Count for the meshed surfaces
VARIANT nMeshedSurface = OSGeometryUI::GetParametricSurfaceCount();
VBA Syntax
' Count for the meshed surfaces
Dim nMeshedSurface As VARIANT = OSGeometryUI.GetParametricSurfaceCount()
